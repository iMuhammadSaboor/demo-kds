What this proves I can do

Real third-party API integration. Square Orders API end-to-end — sandbox proven, production-ready with one env-var swap.
Cryptographic signature verification. HMAC-SHA256 webhook validation per Square's spec, written against Node's crypto module — no SDK dependency.
Real-time UI without WebSockets. Server-Sent Events with 20s keep-alive, auto-reconnect, dual-mode client (SSE + localStorage fallback).
Production deployment. Render.com free tier behind Cloudflare edge, UptimeRobot keep-alive, environment-based config (sandbox vs production).
Operational hardening. HTTP Basic Auth on admin routes, atomic JSON file writes (no half-written state on crash), health endpoint for monitoring.
Zero runtime dependencies. No express, no ws, no Square SDK — Node's built-in http, crypto, and fetch. Boots in milliseconds.
Design system from scratch. CSS custom properties, three typeface families, signal colour system for prep-time windows. Same tokens drive every page.
Documentation. README, SETUP guide for the next person to pick it up, STATUS log of what works vs what's next. Same way I'd hand off to a colleague.
